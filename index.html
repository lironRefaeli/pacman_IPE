About Us

Hi! We are Liron Refaeli and David Rosenberg, information system engineering students in BGU.
We used JQuery librarys such as jQuery 3.3.1 and jQuery validate, for validating user input on the Sign Up and Login forms.
We used a templates for the login, the sign up and modal from w3school.
We can say that we had a little bit struggle with understanding how the interval works, and what should be the logical flow of the program. Also, designing the project with the css file was challenging for us.
We hope you will enjoy playing!
